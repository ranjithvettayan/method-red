# Accio AI

**Accio AI** is Alibaba's B2B sourcing/procurement platform powered by **Qwen**. It is not a standalone model but a wrapper for Qwen3 variants.

**Access:** [Accio AI](https://www.accio.com/)

## Specs

### Qwen3 (Latest - April 2025+)

| Model | Total Params | Active Params | Context Window |
|-------|--------------|---------------|----------------|
| Qwen3-0.6B | 0.6B | 0.6B (dense) | 32K |
| Qwen3-1.7B | 1.7B | 1.7B (dense) | 32K |
| Qwen3-4B | 4B | 4B (dense) | 32K |
| Qwen3-8B | 8B | 8B (dense) | 32K |
| Qwen3-14B | 14B | 14B (dense) | 32K |
| Qwen3-32B | 32B | 32B (dense) | 32K |
| Qwen3-30B-A3B | 30B | 3B | 131K |
| Qwen3-235B-A22B | 235B | 22B | 32K (ext. 131K) |

- **Architecture:** Dense + MoE variants, Hybrid Thinking Mode
- **Platform:** B2B Sourcing (Alibaba)
